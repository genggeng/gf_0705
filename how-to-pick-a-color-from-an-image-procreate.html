How to Pick a Color From an Image in Procreate

If you want to sample a color from a photo, sketch, screenshot, or reference image in Procreate, use the eyedropper gesture. If you also need the exact HEX, RGB, or HSL code for web design, use Image Color AI after sampling the image.

Get HEX from your image

Quick answer

In Procreate, touch and hold on the image where you want to sample a color. The eyedropper appears. Drag it over the exact color, then release to set that color as your active Procreate color.

Step-by-step guide

  1. Open your Procreate artwork or create a new canvas.
  2. Add the image you want to sample, or place it on the canvas as a reference.
  3. Touch and hold on the part of the image that contains the color you want.
  4. When the eyedropper appears, drag to fine-tune the sampled point.
  5. Release your finger or Apple Pencil to select the color.
  6. Use the selected color with Procreate brushes, fills, or palettes.

How to get HEX or RGB from the same image

Procreate is great for painting and sampling colors, but many web and brand workflows need precise values such as HEX, RGB, or HSL. Upload the same image to Image Color AI, click the color, and copy the value from the color panel.

  • Use HEX for CSS, brand guides, and design tools.
  • Use RGB for UI work, JavaScript, and canvas projects.
  • Use HSL when you want to adjust hue, saturation, or lightness.

Procreate color picking tips

Zoom into the image before sampling small details. If a photo has shadows or compression artifacts, sample several nearby points and choose the color that best matches your artwork. For cleaner palettes, use one primary color, one dark support color, one light neutral, and one accent.
